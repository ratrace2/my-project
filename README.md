# Task 7



## Настройка Nginx
- Nginx настроен как обратный прокси для всех сервисов.  
- Маршрутизация:  
  - / → балансировка между фронтенд-контейнерами.  
  - /api/ → backend.  
- Настроен редирект HTTP → HTTPS.  
- SSL-сертификаты помещены в nginx/ssl.  

## Мониторинг и метрики
- Развёрнут Prometheus для сбора метрик с фронтенда, бэкенда и Nginx.  
- Настроены scrape jobs для всех сервисов, чтобы отслеживать доступность и производительность.  

## Логирование
- Развёрнут Loki + Promtail для централизованного сбора логов Docker-контейнеров.  
- Promtail собирает логи всех контейнеров и отправляет их в Loki.  
- Конфигурация источника данных для Grafana добавлена (grafana/provisioning/datasources/loki.yml).
